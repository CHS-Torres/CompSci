🧩 What We Learn (Family-Friendly Unit Guides)

🌐 Web Development (HTML/CSS/JS)

  • Structure & accessibility using semantic HTML
  • Responsive design with CSS
  • Interactive pages with events

How families can help:

  • Ask: “How does your page adapt to different screens?”
  • Review HTML tags together
  • Encourage reading MDN docs

🐍 Python Programming

  • Data types & lists
  • Loops & conditional logic
  • Functions & problem decomposition

How families can help:

  • Ask them to explain a function
  • Encourage checking error messages
  • Help them test one idea at a time

☕ Java (AP CSA)

  • Classes, methods & objects
  • Algorithms and debugging
  • Preparing for AP certification readiness

How families can help:

  • Ask what a class does in their project
  • Discuss debugging steps

🎮 C# & Unity Game Development

  • C# scripting basics
  • Unity components & scenes
  • Game loops, iteration & playtesting

How families can help:

  • Playtest with your student
  • Ask about choices they made in their game
  • Encourage reviewing Unity docs
